TECHNOLOGY

Backend

프로그램 개발의 첫 단계는 프로그래밍 능력
프로그램 개발을 위해서는 프로그래밍 능력이 필연적이며 제가 갖고 있는 대표적인 기술로는
JAVA, JSP, Servlet, Spring MVC
외에도 여러 기술을 보유하고 있습니다.

Frontend

프로그램의 화면 구성은 개발자의 필수적인 기술
Backend에 더불어 Frontend 개발 또한 가능하며 제가 갖고 있는 대표적인 기술로는
JS, JQuery, Ajax, Vue
외에도 여러 기술을 보유하고 있습니다.

Database

실 데이터를 기반으로 프로그램을 구축하는 능력
단순 프로그램을 구축하는 것보다는 실제 데이터로 구축하는 것이 더 효과적인 기술입니다.
Oracle, ERD, MyBatis
등의 기술을 보유하고 있습니다.
